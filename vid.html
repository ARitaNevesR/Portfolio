Sweet Dreams

Sweet Dreams é uma curta-metragem de terror, publicada na revista WayMag

Em Sweet Dreams mostra-se o homicídio de uma família, que é contado no lugar de uma história infantil. Tudo isto porque uma criança se achava crescida demais para ouvir histórias para dormir
